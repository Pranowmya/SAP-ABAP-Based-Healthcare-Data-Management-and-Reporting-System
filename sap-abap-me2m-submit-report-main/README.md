# 🚀 SAP ABAP Report – ME2M Submit Program

## 📌 Overview
This project demonstrates how to create a custom ABAP report that calls the standard SAP ME2M report using the SUBMIT statement.

Instead of recreating the logic, the program executes the standard report and displays the same output.

---

## 🎯 Objective
To build a custom report that:
- Calls standard ME2M report  
- Displays identical output  
- Uses SUBMIT statement effectively  
- Demonstrates reuse of SAP standard programs  

---

## ⚙️ Program Details
**Custom Report:** Z_ME2M_SUBMIT  
**Standard Program Used:** RM06EM00 (ME2M)

---

## 📤 Output

- Same output as ME2M report  
- Standard SAP ALV/List display  
- Includes purchasing document details  

---

## 🔄 Process Flow
1. User enters Material & Plant  
2. Custom report is executed  
3. SUBMIT calls standard program `RM06EM00`  
4. Standard ME2M output is displayed  

---

## 💡 Key Features
- Reuse of standard SAP report  
- No need to rewrite logic  
- Faster development  
- Accurate standard output
